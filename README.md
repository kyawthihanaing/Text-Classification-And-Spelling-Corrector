# NLP Assignment

## Setup
Run `setup.sh` (Linux/Mac) or `setup.ps1` (Windows) to create the virtual environment and project structure.

## Activate Virtual Environment
- Linux/Mac: `source NLP_Assignment/bin/activate`
- Windows: `NLP_Assignment\Scripts\Activate.ps1`

## Install Dependencies
`make setup`

## Format Code
`make fmt`

## Run Tests
`make test`
